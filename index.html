We are learning Github Step by step

    This is Code for Pushing code to repository

  1. git config user.email " Enter email "
  2. git config user.name " Enter name "
  3. git remote add origin "path of git repo"
  4. git init
  5. git status
  6. git add -A
  7. git commit -m "message for commit"
  8. git push origin master

    This is code for pulling code from repository

  1. git clone "repository name / url"
  2. git pull origin master
